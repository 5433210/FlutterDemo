欢迎安装字字珠玑！

字字珠玑是一款专业的书法管理和练习应用程序，专为书法爱好者和专业人士设计。

主要功能：
• 作品管理：轻松管理您的书法作品集
• 字帖练习：提供丰富的练习字帖
• 字符识别：智能识别和分析书法字符
• 多格式支持：支持多种图像和文档格式
• 标签系统：灵活的标签分类系统
• 数据同步：安全的本地数据存储

系统要求：
• Windows 10 版本 1803 (Build 17763) 或更高版本
• 64位操作系统
• 至少 4GB RAM
• 至少 500MB 可用磁盘空间
• 支持DirectX 11的显卡

安装说明：
1. 确保您的系统满足最低要求
2. 关闭所有正在运行的应用程序
3. 以管理员身份运行安装程序
4. 按照安装向导的指示完成安装

如果您在安装过程中遇到任何问题，请访问我们的支持页面或联系技术支持。

感谢您选择字字珠玑！
