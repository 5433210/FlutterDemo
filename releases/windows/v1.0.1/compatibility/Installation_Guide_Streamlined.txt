CharAsGem - Streamlined Installation Package v1.0.1
==================================================

Installation Process:
- Simplified installation wizard
- No information pages during setup
- License agreement only
- Quick and easy installation

System Requirements:
- Windows 7 SP1 and above
- Windows 8/8.1/10/11
- 64-bit architecture

Installation Steps:
1. Double-click CharAsGemInstaller_Legacy_v1.0.1.exe
2. Accept license agreement when prompted
3. Choose installation directory (optional)
4. Select additional options (desktop shortcut, file associations)
5. Click Install to proceed
6. Installation completes automatically

Certificate Installation (Optional):
1. Double-click CharAsGem.cer certificate file
2. Click "Install Certificate"
3. Select "Local Computer"
4. Place certificate in "Trusted Root Certification Authorities"
5. This prevents security warnings

Installation Features:
- Clean, streamlined setup process
- No unnecessary information screens
- Fast installation
- Professional appearance
- Minimal user interaction required
